Hablemos de lo que es HTML

HTML, que significa "Lenguaje de Marcado de Hipertexto" en inglés (Hypertext Markup Language), es el estándar principal para la creación y el diseño de páginas web. Es un lenguaje de marcado que permite estructurar el contenido de una página web mediante el uso de etiquetas y atributos. Las etiquetas describen el significado semántico del contenido, mientras que los atributos proporcionan información adicional sobre las etiquetas.

Para que se usa HTML

HTML (Lenguaje de Marcado de Hipertexto, por sus siglas en inglés) se utiliza para crear y estructurar el contenido de las páginas web. Es el lenguaje fundamental que define la estructura y el significado semántico de los elementos en una página. A continuación, se detallan algunas de las funciones principales de HTML:

Como se estructura una pagina web con HTML

La estructura básica de una página web con HTML sigue un formato estándar. Aquí te proporcionaré una descripción detallada de cada parte de la estructura:

  • Declaración del Tipo de Documento (DOCTYPE): 

					
	<!DOCTYP html>

Esta línea declara la versión de HTML que estás utilizando. En este caso, html es la versión más reciente y comúnmente utilizada.

  • <html lang="es">: Indica el inicio del documento HTML y especifica el idioma de la página. 


	<html lang="es">

  • <head>: Contiene metadatos, enlaces a hojas de estilo (CSS), scripts, y otros elementos que no son parte del contenido principal visible en la página. 


	<head>

  • <title>: Define el título de la página que se muestra en la barra de título del navegador. 


	<title>

  • <body>: Contiene todo el contenido visible de la página. 


	<body>

  • <header>: Puede incluir el logotipo, el menú de navegación, o cualquier elemento de encabezado de la página. 


	<header>

  • <main>: Contiene el contenido principal de la página. 


	<main>

  • <footer>: Puede incluir información de contacto, enlaces adicionales, o cualquier elemento de pie de página. 


	<footer>

Los Encabezados

Los encabezados en HTML son elementos utilizados para definir títulos o secciones dentro de una página web. Proporcionan una estructura jerárquica que ayuda a organizar y dar formato al contenido. Hay seis niveles de encabezados, del <h1> al <h6>, donde <h1> es el nivel más alto y <h6> el nivel más bajo. 


	<h1>: Encabezado principal (título más importante)
	<h2>: Subtítulo
	<h3>: Subtítulo de nivel 3
	<h4>: Subtítulo de nivel 4
	<h5>: Subtítulo de nivel 5
	<h6>: Subtítulo de nivel 6 (título menos importante)

Los parrafos

Los párrafos en HTML se definen mediante el elemento <p>. Este elemento se utiliza para representar un párrafo de texto en una página web. Cuando tienes un bloque de texto que constituye un párrafo lógico o una unidad temática, puedes envolver ese texto dentro de las etiquetas <p> para indicar que es un párrafo. Aquí tienes un ejemplo: 


	<p>Este es un párrafo de ejemplo. Puedes agregar aquí tu contenido de texto.</p>

Los Enlaces

Los enlaces, también conocidos como hipervínculos o simplemente "links", son elementos en HTML que permiten la navegación entre diferentes páginas web o recursos en línea. La etiqueta utilizada para crear enlaces es <a>, y se puede utilizar para vincular a otras páginas, sitios web, archivos, ubicaciones dentro del mismo documento, direcciones de correo electrónico, entre otros. 


	<a href="https://google,com">Click Aqui Para Ir A Google<a>

Las Listas

En HTML, las listas son estructuras que permiten organizar y presentar información de manera ordenada o desordenada. Hay dos tipos principales de listas en HTML: listas ordenadas (<ol>) y listas no ordenadas (<ul>).

  • Listas No Ordenadas (<ul>)

    Las listas no ordenadas se utilizan cuando el orden de los elementos no es importante. Cada elemento de la lista se marca con un marcador, generalmente un punto o un guion. 


	<ul>
    	   <li>Elemento 1</li>
    	   <li>Elemento 2</li>
    	   <li>Elemento 3</li>
	</ul>
