Que es HTML?

HTML (HyperText Markup Language) es el lenguaje estándar utilizado para crear y diseñar páginas web. Es un lenguaje de marcado que permite estructurar el contenido de una página web mediante etiquetas y atributos que definen diferentes elementos, como texto, imágenes, enlaces, formularios, etc.

Cuales Son Los Fundamentos de HTML

Los fundamentos de HTML incluyen los conceptos y elementos básicos que se utilizan para crear y estructurar páginas web. Algunos de los fundamentos clave de HTML son:

Elementos y etiquetas

HTML se basa en el uso de elementos y etiquetas para definir diferentes partes de una página web. Cada elemento comienza con una etiqueta de apertura <nombre_del_elemento> y termina con una etiqueta de cierre </nombre_del_elemento>. Por ejemplo, <p> y </p> se utilizan para crear párrafos de texto.

Atributos:

HTML utiliza atributos para proporcionar información adicional sobre los elementos. Algunos atributos comunes incluyen id (identificador único), class (clase de estilo), href (URL de destino), src (ruta de la imagen), entre otros.

Estructura básica:

Toda página web HTML comienza con el elemento <html> y tiene dos secciones principales: <head> (encabezado) y <body> (cuerpo). El encabezado contiene metadatos, como el título de la página y enlaces a hojas de estilo o scripts, mientras que el cuerpo contiene el contenido visible de la página.

Textos:

HTML permite agregar diferentes tipos de texto, como encabezados (<h1>, <h2>, etc.), párrafos (<p>), listas (<ul>, <o>>, <li>), y otros elementos de texto como negritas (<strong>), cursivas (<em>), subrayados (<u>), entre otros.

Enlaces:

Los enlaces <a> se utilizan para conectar diferentes páginas web entre sí. Pueden contener texto o imágenes y tienen un atributo href que especifica la URL de destino.

Imágenes:

Las imágenes <img> se utilizan para mostrar imágenes en una página web. Tienen un atributo src que especifica la ruta de la imagen y un atributo alt que proporciona un texto alternativo para accesibilidad.

Tablas:

HTML permite la creación de tablas <table> para organizar datos en filas y columnas. Las tablas contienen elementos como <tr> para filas, <td> para celdas de datos y <th> para encabezados de columna o fila.

Formularios:

Los formularios <form> se utilizan para recopilar datos del usuario. Contienen elementos de entrada como campos de texto (<input type="text">), casillas de verificación (<input type="checkbox">), botones (<button>), entre otros.

Que Son Los Encabezados?

Los encabezados en HTML son elementos utilizados para definir los niveles de jerarquía y estructura del texto en una página web. Ayudan a organizar y dar formato al contenido, facilitando su lectura y comprensión. En HTML, hay seis niveles de encabezado, desde el <h1> hasta el <h6>, donde <h1> es el nivel más alto y <h6> es el nivel más bajo.

Aquí hay un resumen de cada uno: 


	<h1>: Este es el encabezado más grande y se utiliza para títulos principales o secciones principales de una página web.
	<h2>: Se utiliza para subsecciones de <h1>, siendo ligeramente menos prominente.
	<h3>: Se utiliza para subsecciones de <h2>, y así sucesivamente hasta <h6>.
	<h4>, <h5>, <h6>: Son encabezados de niveles inferiores, utilizados para subdivisiones más específicas del contenido.

Que son los Parrafos?

Los párrafos en HTML son elementos utilizados para definir bloques de texto que forman unidades coherentes dentro de una página web. Se utilizan para agrupar y estructurar el contenido textual, lo que facilita su presentación y comprensión por parte de los usuarios. En HTML, los párrafos se definen utilizando la etiqueta <p>, que indica el comienzo de un párrafo, y </p>, que indica el final del párrafo.

Por ejemplo: 


	<p>Este es un parrafo </p>
