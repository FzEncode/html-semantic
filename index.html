Que es HTML?

¡Claro! HTML, o HyperText Markup Language (Lenguaje de Marcado de Hipertexto), es el lenguaje estándar utilizado para crear páginas web. Proporciona una estructura básica para el contenido de una página web, permitiendo a los desarrolladores web definir y organizar diferentes tipos de contenido, como texto, imágenes, enlaces, formularios, videos, entre otros. HTML utiliza etiquetas para marcar distintos elementos dentro de una página web y define la relación y estructura entre ellos.

Para que sirve HTML

HTML es como el esqueleto de una página web. Imagina que estás construyendo una casa: el HTML sería como los cimientos y la estructura básica. Te ayuda a decirle al navegador web qué partes componen tu página, como encabezados, párrafos, imágenes, enlaces y más.

Los Fundamentos De HTML

Los fundamentos de HTML incluyen:

  • Elementos y Etiquetas:

    HTML utiliza elementos y etiquetas para definir la estructura y el contenido de una página web. Los elementos se definen mediante etiquetas de apertura <tag> y etiquetas de cierre </tag>. Por ejemplo, <p> define un párrafo.

  • Atributos:

    Los atributos proporcionan información adicional sobre los elementos y se especifican dentro de las etiquetas de apertura. Por ejemplo, el atributo src se utiliza en la etiqueta <img> para especificar la ubicación de la imagen.

  • Estructura básica:

    Una página HTML típica comienza con <!DOCTYPE html> seguido de <html>, que envuelve todo el contenido de la página. Luego, dentro de <html>, encontramos <head> para metadatos y <body> para el contenido visible.

  • Texto:

    HTML permite incluir texto dentro de los elementos, ya sea directamente o mediante el uso de etiquetas de texto como <p>, <h1> para encabezados, <strong> para texto en negrita, <em> para texto en cursiva, entre otros.

  • Enlaces:

    Los enlaces se crean utilizando la etiqueta <a>. El atributo href dentro de la etiqueta <a> especifica la URL a la que se enlaza el texto.

  • Imágenes:

    Para incrustar imágenes en una página web, se utiliza la etiqueta <img>. El atributo src especifica la ruta de la imagen.

  • Listas:

    HTML permite crear listas ordenadas (<ol>) y no ordenadas (<ul>). Los elementos de la lista se definen utilizando la etiqueta <li>.

  • Tablas:

    Las tablas se crean utilizando las etiquetas <table>, <tr> para filas y <td> para celdas de datos.

  • Formularios:

    Los formularios permiten a los usuarios enviar datos al servidor. Se crean utilizando la etiqueta <form>, y dentro de ella, se utilizan etiquetas como <input> para campos de entrada, <select> para listas desplegables, <textarea> para áreas de texto, etc.

Como esta estructura una pagina web con HTML

Una página web estructurada con HTML sencillo y semántico podría tener la siguiente estructura: 


<!DOCTYPE html>
<html lang="es">
<head>
    	<meta charset="UTF-8">
    	<meta name="viewport" content="width=device-width, initial-scale=1.0">
    	<title>Título de la Página</title>
</head>
<body>
    <header>
        <h1>Logo o Título Principal</h1>
        <nav>
            <ul>
                <li><a href="#inicio">Inicio</a></li>
                <li><a href="#contact">Contacto</a></li>
            </ul>
        </nav>
    </header>

    <main>
        <section id="inicio">
            <h2>Sección de Inicio</h2>
            <p>Bienvenido a mi página web.</p>
        </section>

        <section id="acerca">
            <h2>Acerca de</h2>
            <p>Información sobre mí o sobre mi empresa.</p>
        </section>
    </main>

    <footer>
        <p>Derechos de autor © 2024 Mi Empresa</p>
    </footer>
</body>
</html>

Los Encabezados

En HTML, los encabezados son elementos utilizados para definir la estructura jerárquica y visualmente destacada del contenido de una página web. Estos elementos van desde <h1> hasta <h6>, donde <h1> es el encabezado más importante y <h6> el menos importante. Aquí tienes una breve descripción de cada uno:

  • <h1>

    Este es el encabezado más importante y generalmente se utiliza para el título principal de la página. Debe haber solo un <h1> por página, y este título suele describir el tema principal o el propósito de la página.

  • <h2>

    Es el siguiente nivel de importancia después de <h1>. Se usa para subsecciones importantes del contenido. Puede haber varios <h2> en una página, cada uno representando una sección principal dentro del tema general de la página.

  • <h3> a <h6>

    Estos son encabezados de niveles inferiores, donde <h3> es el más importante dentro de este grupo y <h6> el menos importante. Se utilizan para subdividir las secciones representadas por los encabezados superiores.

Los Parrafos

En HTML, los párrafos se definen mediante la etiqueta

. Esta etiqueta se utiliza para marcar bloques de texto que forman un párrafo en el contenido de una página web. Los párrafos son elementos semánticos que ayudan a estructurar el texto y mejorar la legibilidad de una página. 


	<p>Este es un párrafo de texto. Los párrafos son útiles para agrupar y estructurar el contenido de una página web.</p>

Los Enlaces

En HTML, los enlaces se definen utilizando la etiqueta <a>, que significa "ancla". Los enlaces permiten a los usuarios navegar de una página web a otra página web, a otra sección de la misma página web, o a recursos externos como archivos PDF, imágenes, videos, etc. También se pueden usar para enviar datos a través de formularios o para ejecutar scripts.


Aquí tienes un ejemplo básico de cómo se usa la etiqueta para crear un enlace:
